<h3>
# University Network Topology Project<br>
# Author: Sayfullah<br>
# Description: This project simulates a multi-campus network setup using Cisco Packet Tracer.<br>
# Includes VLAN segmentation, inter-campus routing, and server integration.<br>
## Routing Protocol RIP v2 is configured on routers to enable dynamic route exchange between campuses.<br>
##Inter-VLAN RoutingRouter-on-a-stick configuration ensures devices in different VLANs can communicate.<br>
##DHCP is configured on the routers to automatically assign IP addresses to all VLANs.<br>
