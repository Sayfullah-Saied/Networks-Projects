<h3>start-up company topology</h3>

<h4>
  1- the company has 3 departments (IT - HR - Customer Service)<br>
  2- each department has its own vlan and inter-vlan routing enabled on the router<br>
  3- dhcp server enabled on the router with 3 pools one for each department, hosts will be assigned IPs automatically <br>
  4- access point in each department to provide wireless network for users
</h4>
